📊 SMARTPHONE USAGE BEHAVIOR ANALYSIS DASHBOARD – POWER BI
🧠 The Story Behind This Project

Smartphones have become an integral part of everyday life, influencing how people communicate, learn, work, and entertain themselves. However, the rapid increase in smartphone usage has also raised concerns about excessive screen time and changing digital habits.
Instead of simply creating charts about mobile usage, this project aims to explore a deeper behavioral question: How do demographic factors such as age, gender, and location influence smartphone usage patterns?
Understanding these patterns is important because smartphone usage behavior affects productivity, digital wellbeing, and lifestyle balance. By analyzing user behavior through data visualization, this project provides insights into how different groups interact with mobile applications and how usage habits vary across populations.
Through this dashboard, I attempt to analyze smartphone usage trends and identify the key factors that influence screen time and application usage patterns.

🔍 What This Dashboard Explores

This Power BI dashboard analyzes smartphone usage behavior using a dataset containing demographic and mobile usage information.
The dashboard explores questions such as:
How does daily screen time vary across different age groups?
Which application categories (social media, gaming, productivity) dominate smartphone usage?
How does smartphone usage differ based on gender and location?
Does the number of installed apps influence total screen time?
Which demographic factors contribute the most to high smartphone usage patterns?
The dashboard includes interactive filters and visualizations that allow users to explore smartphone usage patterns dynamically.

📈 Key Insights from the Data

After analyzing the dataset, several interesting patterns were identified:
Screen Time Patterns: Young adults tend to have the highest average screen time, indicating greater dependence on smartphones.
Application Usage Distribution: Social media applications account for the largest share of app usage, followed by gaming and productivity apps.
App Count Influence: Users who have a higher number of installed apps generally show higher screen time usage.
Demographic Differences: Smartphone usage patterns vary across gender and geographic locations, indicating behavioral differences among users.
These insights help understand how smartphone usage behavior differs across demographic groups.
